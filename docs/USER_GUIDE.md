#   Travel Content Factory — Полная инструкция

> От установки до первого смонтированного ролика

---

##   Шаг 1: Установка

### Системные зависимости

Открой терминал и установи FFmpeg и exiftool:

```bash
brew install ffmpeg exiftool
```

Проверь, что всё встало:

```bash
ffmpeg -version   # должно показать версию, например 7.x
exiftool -ver     # должно показать версию, например 13.x
```

### Python-окружение

```bash
cd ~/travel-content-factory
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### API-ключ DeepSeek (для AI-функций)

1. Зарегистрируйся на [platform.deepseek.com](https://platform.deepseek.com)
2. Пополни баланс (от $1) и создай API-ключ
3. Пропиши его:

```bash
cp .env.example .env
# отредактируй .env:
# DEEPSEEK_API_KEY=*** 
```

### Запуск

```bash
./start.sh
```

Открывай **http://localhost:8000**

##   Шаг 2: Загрузка медиа в архив

У тебя есть папка с фото и видео из путешествия. Например:

```
~/Photos/Paris-2024/
    IMG_0001.JPG     ← фото с Эйфелевой башней
    IMG_0002.MOV     ← видео из Лувра
    IMG_0003.JPG     ← круассан
    IMG_0004.MP4     ← прогулка по Монмартру
    ...
```

### Способ 1: Scan Folder

1. В приложении открыта вкладка **Archive**
2. Жми кнопку **Scan Folder**
3. Введи путь: `/Users/me/Photos/Paris-2024`
4. Жми OK

Приложение рекурсивно обойдёт папку, извлечёт EXIF (дата, GPS, размер, длительность для видео) и покажет всё в сетке:

```
┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│   IMG_0001   │ │  IMG_0002    │ │   IMG_0003   │ │  IMG_0004    │
│  [превью]    │ │  [превью]    │ │  [превью]    │ │  [превью]    │
│  France 2024 │ │  France 2024 │ │  France 2024 │ │  France 2024 │
│  photo       │ │  video       │ │  photo       │ │  video       │
└──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘
```

### Способ 2: Upload Files

1. Жми **Upload Files**
2. Выбери файлы в диалоге
3. Файлы копируются в `uploads/` и индексируются

### Фильтрация

После загрузки можно фильтровать:
- **Type:** All types / Photos / Videos
- **Country:** появится список стран из GPS-данных
- **Year:** список годов съёмки
- **Hashtag:** поиск по названиям файлов (пока ручные хэштеги не поддерживаются)

### Три режима

####   Auto — автоматический

1. Выбери **Auto — system selects clips**
2. Задай страну (France) и год (2024)
3. Укажи длительность (например, 30 секунд)
4. Жми **Create Project**

Система сама найдёт все видео из France 2024, нарежет по N секунд и составит список клипов.

####   Manual — ручной

1. Выбери **Manual — pick clips yourself**
2. Жми **Create Project** (клипы добавим отдельно)
3. Сейчас через API: `POST /api/projects/{id}/clips` с массивом `{"clips": [{"media_id": 1, "start_time": 0, "duration": 5}, ...]}`

>   В будущем: drag & drop клипов прямо в интерфейсе

####   By Script — по сценарию

1. Выбери **By Script — AI breaks into scenes**
2. Вставь текст сценария (появится поле Script Text):

```
Париж просыпается. Солнце золотит крыши.
Круассан и кофе в уличном кафе.
Прогулка вдоль Сены. Эйфелева башня в лучах заката.
```

3. Жми **Create Project**

AI (DeepSeek) разобьёт сценарий на сцены, и для каждой подберёт подходящие клипы по GPS и дате.

##   Шаг 4: Рендер и экспорт

После создания проекта в списке появится карточка:

```
┌──────────────────────────────────────────────────────────┐
│ Paris Morning Reel                                       │
│ manual · tiktok · 6 clips · 30s  DRAFT                   │
│                                    [Render]  [Del]       │
└──────────────────────────────────────────────────────────┘
```

1. Жми **Render** — FFmpeg склеит клипы
2. Статус изменится с DRAFT → PROCESSING → READY
3. Появится кнопка **Download**

Жми **Download** → браузер скачает готовый `.mp4`.

Файл также лежит в папке `exports/project_N_Paris_Morning_Reel.mp4`.

###   Наложение музыки

Брось свои `.mp3` треки в папку `music/` и передай путь при рендере:

```bash
curl -X POST http://localhost:8000/api/projects/1/render \
  -H 'Content-Type: application/json' \
  -d '{"music_path": "music/chill-lofi.mp3", "music_volume": 0.3}'
```

Что происходит внутри:
```
1. concat     → склеиваем клипы
2. overlay    → loop music на всю длину, fade in 1.5s, fade out 2s
3. normalize  → loudnorm до -14 LUFS (стандарт соцсетей)
4. export     → готовый .mp4 с музыкой и выровненной громкостью
```

- **music_volume:** от 0.1 (еле слышно) до 0.5 (музыка перебивает голос)
- **Форматы:** `.mp3`, `.m4a`, `.wav`, `.ogg`, `.flac`
- **Loop:** короткий трек автоматически зациклится на всё видео
- **Fade:** плавное появление (1.5s) и затухание (2s) музыки

Где взять бесплатную музыку:
- [Pixabay Music](https://pixabay.com/music/)
- [Uppbeat](https://uppbeat.io/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)
- [Mixkit](https://mixkit.co/free-stock-music/)

### Что если ошибка?

- **DRAFT → ERROR:** нет клипов или FFmpeg не может прочитать исходники
- **FFmpeg error:** проверь, что исходные файлы не удалены и доступны по пути
- **AI error (401):** проверь `DEEPSEEK_API_KEY` в `backend/.env`

##   Типичные сценарии использования

### Сценарий А: «Вернулся из Италии, хочу рилс»

1. Отсканируй папку `~/Photos/Italy-2024`
2. Отфильтруй: Type=Videos, Country=Italy
3. Content Factory → тема: «10 дней в Италии за 30 секунд»
4. Скопируй сценарий → Projects → New Project (By Script)
5. Вставь сценарий → Render → Download
6. Скопируй хэштеги и подпись из Content Factory
7. Залей видео + подпись в Instagram

### Сценарий Б: «Много фото из разных стран, хочу разобрать»

1. Отсканируй корневую папку `~/Photos/`
2. Фильтруй по странам: France, Italy, Japan, ...
3. Фильтруй по годам: 2022, 2023, 2024
4. Быстро находи нужный контент для монтажа

### Сценарий В: «Ручной контроль над каждым кадром»

1. Создай Manual-проект
2. Через API задай клипы с точными таймкодами
3. Render
4. (В будущем: drag & drop клипов в UI)

##   Советы

- **Без GPS — не страшно.** Файлы без координат просто не попадут в фильтр по стране, но в гриде видны
- **FFmpeg на CPU.** На MacBook Pro 2015 рендер 30-секундного ролика — ~10-30 секунд
- **DeepSeek экономичный.** Генерация одного сценария — < $0.01
- **База SQLite.** Файл `data/travel_factory.db` можно бекапить и переносить
- **Чистка базы.** Удали `data/travel_factory.db` — при следующем запуске создастся заново

##   Решение проблем

| Проблема | Причина | Решение |
|---|---|---|
| «exiftool not found» | Не установлен exiftool | `brew install exiftool` |
| «ffmpeg not found» | Не установлен FFmpeg | `brew install ffmpeg` |
| «unable to open database» | Нет прав на `data/` | `mkdir -p data` |
| AI возвращает 401 | Неверный ключ DeepSeek | Проверь `DEEPSEEK_API_KEY` в `backend/.env` |
| AI возвращает 402 | Кончился баланс | Пополни на platform.deepseek.com |
| Рендер ERROR | Битые исходные файлы | Проверь пути: `SELECT original_path FROM media_files` |
| Страна не определяется | Нет GPS в EXIF | Нормально, фильтр по стране просто пустой |
| Порт 8000 занят | Другой процесс | `lsof -i :8000` → `kill` или смени порт в `.env` |
